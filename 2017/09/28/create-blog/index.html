Hexo相关配置

Hexo

几个主要命令

  • hexo new post/draft <file-name>

会在source目录下的_posts或则_drafts下生成file-name.md。
也就是说你在使用hexo new创建文件的时候，不需要带文件扩展名，默认生成markdown文件

  • hexo clean

清楚缓存，一般在使用hexo deploy命令时，先执行该命令。

  • hexo deploy

推送到仓库到远程
