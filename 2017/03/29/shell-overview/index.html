  • C Shell (/usr/bin/csh)
  • K Shell (/usr/bin/ksh)

运行Shell脚本的2种方法

  1. 作为可执行程序

  2. 作为解释器参数

这种方式运行的脚本，不需要再第一行指定解释器的信息，即使指定了也没用
